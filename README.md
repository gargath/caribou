# Caribou - An Exercise in Infrastructure-as-Code

##Requirements
* AWS SDK for Ruby v2
* NetAddr

* Uses Akamai's public IP service (http://whatismyip.akamai.com)


##Notes
* Currently uses Aws.use_bundled_cert! for an easy way to work on Windows.
